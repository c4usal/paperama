# I think the killer feature isn't on this list.

Everything above still assumes the scientist asks questions.

What if Paperama watches the global knowledge graph continuously and *proactively* alerts researchers?

For example:

> "Your lab studies kinase A. Overnight, three unrelated papers in cancer biology, structural biology, and immunology collectively increased the probability that kinase A interacts with pathway B. No one has published this connection."

No search.

No prompt.

The system detects an **emerging discovery**.

---

I think that's where the real leverage is.

Most tools help you retrieve known knowledge.

Paperama should help you notice **newly possible knowledge**.

That's a capability that directly narrows the gap between existing scientific knowledge and the next scientific discovery. It shifts the product from being reactive ("find me papers") to proactive ("here's a hypothesis that only became visible because the global body of evidence changed").
